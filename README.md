# VicBlog Front-End Web App

This is the front-end part of VicBlog.

## Features
- React+Redux+React Router
- TypeScript
- Webpack 2
- Hot Module Replacement

## Development

1. `npm install`
2. `npm start`
3. Open `http://localhost:8080`
4. Now you can enjoy the hot module replacement :)
